# Match-Three
Windows Game <br>
Built Using Unity 2020.3.2f1 <br>
Dilo Game Academy's Assignment

## Player Control
<kbd>Esc</kbd> Close Application <br>
<kbd>R</kbd> Restart the Game <br>
Left Klik for switch tile<br>

## Gameplay
You must match 3 same color to get score
